## Tool index

| Tool | What it does |
|------|-------------|
| [`check_balance`](#check_balance) | Get current USDC balance and treasury info |
| [`get_agents`](#get_agents) | List all agents linked to this owner |
| [`get_policy`](#get_policy) | Read the spending policy for an agent |
| [`set_policy`](#set_policy) | Update daily budget or per-payment limits |
| [`get_spend_history`](#get_spend_history) | List recent payments made by this owner |
| [`allocate_budget`](#allocate_budget) | Move USDC from treasury into an agent's wallet |

| [`list_models`](#list_models) | List available Float models with pricing |
| [`float_pay`](#float_pay) | Send USDC from an agent's wallet to any address |

## check_balance

Returns the current USDC balance for the owner's treasury wallet.

**When to call:** Before taking on any paid work. Check that funds are available before proceeding.

**Parameters:** None

**Example output:**
```json
{
  "ownerId": "owner_abc",
  "treasuryWalletAddress": "UXeH32...VL",
  "amount": "2485000",
  "decimals": 6,
  "currency": "USDC",
  "formattedBalance": "24.85 USDC"
}
```

## get_agents

List all agents linked to this owner, including their status, wallet address, and spending policy.

**When to call:** When you need to pick an agent for a task, check which agents are active, or find an agent's wallet address.

**Parameters:** None

**Example output:**
```json
{
  "ownerWallet": "GLn3Dq...BB",
  "agents": [
    {
      "agentId": "research-agent",
      "displayName": "Research Agent",
      "walletAddress": "4PuqVr...fe",
      "status": "active",
      "policy": {
        "dailyBudget": "5.00",
        "perTxLimit": "1.00"
      },
      "lifetimeSpent": "12.40",
      "lifetimePayments": 88
    }
  ]
}
```

**Status values:** `active` — running normally. `killed` — suspended, no payments allowed.

## get_policy

Read the current spending policy for an agent — daily budget, per-transaction limit, and allowed payment rails.

**Parameters:**

| Name | Type | Description |
|------|------|-------------|
| `agentId` | string | The agent ID to inspect |

**Example output:**
```json
{
  "agentId": "research-agent",
  "policy": {
    "dailyBudget": "50.00",
    "perTxLimit": "5.00",
    "killOnBudget": false,
    "blockUnknown": false,
    "allowX402": true,
    "allowMpp": false
  }
}
```

---

## set_policy

Update the spending policy for an agent. All fields are optional — only the fields you pass are changed.

**Parameters:**

| Name | Type | Description |
|------|------|-------------|
| `agentId` | string | The agent to update |
| `dailyBudget` | string | Max USDC the agent can spend in 24h (e.g. `"50"`) |
| `perTxLimit` | string | Max USDC per single payment (e.g. `"2"`) |
| `killOnBudget` | boolean | Auto-suspend the agent when daily budget is exhausted |
| `blockUnknown` | boolean | Reject payments to unrecognized destinations |
| `allowX402` | boolean | Allow HTTP 402 micropayments |
| `allowMpp` | boolean | Allow MPP protocol payments |

**Example:**
```
set_policy({ agentId: "research-agent", dailyBudget: "25.00", perTxLimit: "2.00" })
```

## get_spend_history

Fetch recent payment activity across all agents for this owner.

**Parameters:**

| Name | Type | Description |
|------|------|-------------|
| `limit` | number | Max records to return (default: 20, max: 100) |

**Example output:**
```json
{
  "ownerWallet": "GLn3Dq...BB",
  "count": 2,
  "history": [
    {
      "agentId": "research-agent",
      "amount": "0.06",
      "currency": "USDC",
      "recipient": "5EodzV...wX",
      "timestamp": "2026-04-22T10:30:00Z"
    }
  ]
}
```

## allocate_budget

Transfer USDC from the owner's treasury into a specific agent's wallet so it can spend.

**Parameters:**

| Name | Type | Description |
|------|------|-------------|
| `agentId` | string | The agent to fund |
| `amount` | string | USDC amount to transfer (e.g. `"5"` or `"0.50"`) |

**Example output:**
```json
{
  "agentId": "research-agent",
  "amount": "5000000",
  "currency": "USDC",
  "txSignature": "5Qv...abc"
}
```

<Warning>
This moves real funds on-chain. The Solana transaction is irreversible. Confirm the agent ID and amount before calling.
</Warning>

## kill_agent

Suspend an agent immediately. All payments are blocked. The wallet and balance are preserved — nothing is lost.

**Parameters:**

| Name | Type | Description |
|------|------|-------------|
| `agentId` | string | The agent to suspend |

**Example output:**
```json
{
  "agentId": "research-agent",
  "status": "killed"
}
```

Use [`resume_agent`](#resume_agent) to re-enable it.

---

## resume_agent

Re-activate a suspended agent so it can make payments again.

**Parameters:**

| Name | Type | Description |
|------|------|-------------|
| `agentId` | string | The agent to resume |

**Example output:**
```json
{
  "agentId": "research-agent",
  "status": "active"
}
```

## rename_agent

Change an agent's display name. The `agentId` used in API calls stays the same — only the human-readable label changes.

**Parameters:**

| Name | Type | Description |
|------|------|-------------|
| `agentId` | string | The agent to rename |
| `displayName` | string | New display name |

## list_models

List all image generation models available on Float with pricing per resolution. Call this before generating an image to show the user what's available.

**Parameters:** None

**Example output:**
```json
{
  "models": [
    {
      "id": "nano-banana-pro",
      "displayName": "Nano Banana Pro",
      "description": "Photorealistic image generation. Best quality.",
      "cost1k": "0.14",
      "cost2k": "0.17",
      "cost4k": "0.17"
    },
    {
      "id": "nano-banana-2",
      "displayName": "Nano Banana 2",
      "description": "Fast, accurate image generation with strong text rendering.",
      "cost1k": "0.06",
      "cost2k": "0.08",
      "cost4k": "0.11"
    }
  ],
  "count": 2
}
```

**Resolution options:** `1K` (standard), `2K` (high quality), `4K` (highest quality)  
**Aspect ratio options:** `1:1` (square), `16:9` (landscape), `9:16` (portrait)

For a complete image generation walkthrough, see [Image Generation](/guides/image-generation).

## float_pay

Send USDC directly from an agent's wallet to any Solana address. Returns a transaction signature confirming the payment.

**Parameters:**

| Name | Type | Description |
|------|------|-------------|
| `recipient` | string | Solana wallet address to send USDC to |
| `amount` | string | USDC amount (e.g. `"1.50"` or `"10"`) |
| `currency` | string | Currency — currently `"USDC"` only (optional, defaults to USDC) |
| `memo` | string | Optional note about what this payment is for |

**Example output:**
```json
{
  "success": true,
  "agentId": "research-agent",
  "agentWalletAddress": "4PuqVr...fe",
  "recipientAddress": "5EodzV...wX",
  "amount": "1500000",
  "currency": "USDC",
  "txSignature": "5Qv...abc",
  "solscanUrl": "https://solscan.io/tx/5Qv...abc"
}
```
